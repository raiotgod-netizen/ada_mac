# macOS: dependencias de sistema (requiere Homebrew)
# Instalar ANTES de pip install -r requirements.txt
# ─────────────────────────────────────────────────────────────
#
# 1) Xcode tools:
#    xcode-select --install
#
# 2) Homebrew (si no está):
#    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
#
# 3) Dependencias de sistema para audio/video:
#    brew install portaudio ffmpeg
#
# 4) Luego instalar Python deps:
#    pip install -r requirements.txt
#
# 5) Las dependencias adicionales (PyPDF2, watchdog, nvidia-ml-py):
#    pip install PyPDF2 watchdog nvidia-ml-py
#
# ─────────────────────────────────────────────────────────────
# NOTA: pyaudio requiere portaudio de sistema.
# Si pip install pyaudio falla, brew install portaudio primero.
